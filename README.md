# Student Management System

This is a simple Java console-based Student Management System.

## Features
- Add student
- Delete student
- Search student
- Uses ArrayList to store student data
- Follows Object-Oriented Programming concepts

## Technologies Used
- Java
- ArrayList
- OOP concepts

## How it works
The system allows users to manage student records by adding, searching,
and deleting students using their ID.

This project is created for learning Java and basic data structures.
