# Student Management System

## Description
A Java-based console application to manage student records.
The system supports adding, searching, and deleting students using their ID.
It follows Object-Oriented Programming principles and includes input validation
and exception handling.

## Features
- Add student
- Search student by ID
- Delete student
- Input validation
- Exception handling
- Uses ArrayList for data storage

## Technologies Used
- Java
- Collections (ArrayList)
- OOP concepts

## How It Works
- User selects an option from the menu
- Input is validated using try-catch
- Business logic is handled in a service layer
- Errors like invalid input or missing student are handled gracefully

## Enhancements
- Input validation using Scanner and exceptions
- Duplicate student ID prevention
- Service-layer refactoring
- Clean and readable code structure

## How to Run
1. Compile all `.java` files
2. Run `StudentManagementSystem.java`
